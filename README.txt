Project setup:


1. Print out 4 ArUco markers from https://chev.me/arucogen/. Change ids from 0-3. 
2. Arrange markers in order: 0-- top left, 1-- top right, 2--bottom right, 3--bottom left.
3. Run realistic_ar_key.py
4. Aim webcam at markers and on your screen you should see the keyboard projection.
5. Move index finger slowly across keyboard towards intended key and tap index finger down quickly in key cell that you want to detect.
